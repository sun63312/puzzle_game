# 3关益智游戏 说明文档

## 🎮 游戏介绍

这是一个包含3个不同类型益智关卡的小游戏，使用Python Tkinter GUI开发，无需任何第三方依赖即可运行。

### 游戏关卡
1. **🧠 记忆翻牌** - 找出所有配对的卡片，训练记忆力
2. **🧩 滑块拼图** - 经典3x3数字滑动拼图
3. **🔢 4x4数独** - 填写数字1-4，保证每行每列不重复

## ▶️ 运行方式

### 方法1: 直接运行EXE
直接双击 `dist/puzzle_game.exe` 即可启动游戏，不需要安装任何软件。

### 方法2: 运行Python源代码
```bash
python puzzle_game.py
```

> 要求Python 3.7+，不需要安装额外库，使用Python内置tkinter。

---

## 🛠️ 打包为EXE的方法

使用PyInstaller将Python代码打包为独立EXE文件：

1. 安装PyInstaller:
```bash
pip install pyinstaller -i https://pypi.tuna.tsinghua.edu.cn/simple
```

2. 执行打包命令:
```bash
pyinstaller --onefile --windowed puzzle_game.py
```

### 参数说明:
- `--onefile`: 打包为单个EXE文件
- `--windowed`: 不显示控制台窗口（GUI程序专用）

打包完成后，EXE文件会生成在 `dist/` 目录下。

## 🧠 代码运行原理

### 整体架构
```
PuzzleGame 主类
├── 窗口初始化与大小设置
├── 关卡管理与切换逻辑
└── 3个关卡实现
    ├── MemoryLevel 记忆翻牌
    ├── SlidePuzzleLevel 滑块拼图
    └── SudokuLevel 数独游戏
```

### 核心设计模式
1. **继承模式**: 所有关卡都继承自 `BaseLevel` 基类，统一了显示/隐藏接口
2. **回调模式**: 每个关卡完成后调用回调函数自动进入下一关
3. **事件驱动**: 所有用户操作通过Tkinter事件系统处理

### 各关卡实现原理

#### ✅ 记忆翻牌关卡
- 生成6对共12个随机打乱的卡片
- 每次最多翻开2张卡片
- 500ms后自动判断是否配对
- 配对成功则卡片变为绿色不可点击
- 全部配对成功后通关

#### ✅ 滑块拼图关卡
- 3x3网格包含数字1-8和一个空格
- 点击空格旁边的数字可以滑动
- 每次移动后检查是否按顺序排列
- 达到12345678顺序时通关

#### ✅ 数独关卡
- 随机生成合法的4x4数独答案
- 随机隐藏8个数字供用户填写
- 检查时验证每行每列数字不重复
- 全部填写正确后通关

|------|----------|
